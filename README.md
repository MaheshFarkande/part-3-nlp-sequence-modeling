# Task 3: Why Text Must Be Converted Into Vectors
Machine learning and deep learning algorithms are fundamentally mathematical engines. They apply weights, biases, matrix multiplications, and calculus optimization techniques (like gradient descent) to data inputs.Text strings like "unreliable service" possess no intrinsic geometric properties or numeric dimensions. Vectorization maps words or characters into high-dimensional numeric spaces. This step encodes linguistic properties, structural syntax, or conceptual meanings into numeric matrices that algorithms can calculate, compare, and optimize against.

# Task 5: Sequence Model Architectural Design & Input Flow
Input Sequence: Text is converted to standard integers using a vocabulary mapping matrix, then padded or truncated to a consistent sequence length of $T = 20$. The input tensor shape passing into the network is $\text{(Batch Size, 20)}$.Embedding Layer: Maps each token index into a dense continuous numeric vector space of size $D = 64$. It morphs discrete indices into a spatial matrix of dimensions $\text{(Batch Size, 20, 64)}$.Recurrent/Sequence Layer (LSTM): Processes vectors sequentially across 20 time steps. The memory cells pass a hidden state vector $h_t$ and cell state vector $c_t$ sequentially forward, calculating semantic dependencies over time steps. It outputs a singular context summary tensor of shape $\text{(Batch Size, 64)}$.Output Layer: Uses a Fully Connected Dense network with a Softmax activation layer to output a normalized probability distribution across 3 discrete classes (Negative, Neutral, Positive).Loss Function: Sparse Categorical Crossentropy is used. It mathematically penalizes deviations between the target true index integer ($0, 1, \text{or } 2$) and the output probability distribution vectors.Evaluation Metric: Accuracy is monitored globally to track the proportion of perfectly matched customer sentiment labels against predictions.

# Task 6: Attention and Transformer Reflection
## 1. Why RNNs Struggle with Long-Term Dependencies
Recurrent Neural Networks process text sequentially, token-by-token.  As sentence strings expand in length, information must flow sequentially through continuous matrix calculations. During backpropagation over long step sequences, gradients are repeatedly multiplied by weight matrices. This math forces gradients to either shrink exponentially (Vanishing Gradients) or scale out of control (Exploding Gradients), wiping out context from the earliest tokens.

## 2. How LSTMs Help with Memory
Long Short-Term Memory units prevent information loss by integrating a specialized structural "Cell State" running through the layer sequence, regulated by internal neural network structures called Gates:Forget Gate: Drops irrelevant context from previous steps.Input Gate: Writes fresh critical contextual data from the current token step.Output Gate: Determines what historical data gets surfaced forward to the next hidden layer state.This gate infrastructure provides an active information highway, preventing earlier gradients from vanishing completely during training.

##   3. What Attention Solves in Sequence-to-Sequence Tasks
Even with gates, traditional encoder-decoder architectures create an operational bottleneck: they force an entire source input phrase to compress into one fixed-length vector before decoding. Attention bypasses this limitation. It creates direct algorithmic shortcuts between all historical inputs and output states, allowing the decoder to look back at the entire input sequence and focus heavily on specific, relevant words regardless of distance.

## 4. Why Transformers are Vital to Modern Generative AI
Transformers discarded recurrent sequential loops entirely in favor of Self-Attention mechanisms operating in parallel.Parallel Execution: Because inputs are parsed simultaneously rather than step-by-step, networks utilize modern GPU clusters at maximum efficiency, unlocking massive scale.Global Context: A token calculates context relative to every other token instantly, creating a deep understanding of syntax and intent.This structural shift paved the way for massive foundational pre-trained Large Language Models (like GPT, Claude, and Gemini) capable of high-fidelity generation, robust logical reasoning, and long-form document comprehension.
